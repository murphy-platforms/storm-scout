# Example CSV Datasets

These CSV files are **example data only** — fictitious facility names at publicly known city locations. They are not based on any real organization's operations or infrastructure.

Use these as format references when preparing your own location CSV for import via `import-offices.js`. See the main [README](../../../../README.md#customize-for-your-organization) for import instructions.

| File | Description |
|------|-------------|
| `airports.csv` | US airports (from OpenStreetMap) |
| `data-centers.csv` | Fictitious data center locations |
| `drive-in-theaters.csv` | US drive-in theaters (from OpenStreetMap) |
| `logistics-hubs.csv` | Fictitious distribution/port facilities |
| `ranger-stations.csv` | US National Forest ranger stations (from OpenStreetMap) |
| `retail-chain.csv` | Fictitious retail store locations |
| `train-stations.csv` | US train stations (from OpenStreetMap) |
